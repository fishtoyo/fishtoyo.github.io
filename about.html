關於我

擁有16年以上實務經驗

- 前端開發： HTML5, CSS3, SCSS, RWD, JavaScript, TypeScript
- 框架與工具： Vue, React, Tailwind CSS, GSAP, jQuery
- 設計與實作： 1:1 Pixel-Perfect, UI Design, Figma, PS/AI, Wireframe
- 技術優化： SEO優化, 網頁效能優化, 跨瀏覽器相容性

自 13 歲製作第一個網站起，這份熱愛已持續 26 年，早已內化成我生活的一部分。
憑藉這份初衷，我一路攻讀資料處理與資訊管理專業，並正式投入網頁開發領域 16 年。

我始終將不斷進化的技術視為養分，從早期的 jQuery、Bootstrap 到現代的 Vue, React 與 AI 協作開發，我享受每一場技術變革帶來的可能性。

在累積 300+ 專案的過程中，磨練出對跨裝置適應性 RWD 的掌控，對於 UI 精準判斷與自訂模組化元件來提高效率。

不論是活動網站、購物車平台、企業形象網站，或是 舊站 RWD 重構，我都具備豐富經驗與實作成果。對於跨瀏覽器、跨裝置的適應性以及人性化的互動設計，亦有成熟的理解與應用。

網站設計對我而言不只是專業，更是結合熱情與創造力的持續探索。
