大型 B2B 專案實績
台灣商業媒合平台

網址：https://www.taiwanbm.com.tw/
一人操刀製作超過 150 頁規模的大型 B2B 電商系統。在高度複雜度的使用者、會員及商家管理介面中，展現了表單邏輯整合與 UI 實作能力。
技術架構與模組化
採用 HTML5 / CSS3 (SSR 架構) 為核心，並自定義 SCSS 模組化元件庫。透過組件化思維，在面對龐大頁面量時仍能維持一致的視覺規範，並大幅提升開發效率與後續維護的便利性。
高複雜度介面挑戰
成功克服 「五層垂直分類主選單」 的實作難題。在確保導覽結構清晰的前提下，與企劃及工程師深度溝通，規劃出兼具操作直覺與效能的選單功能，完美呈現龐大的資訊階層。
效能與搜尋優化
精準管理 JavaScript / jQuery 輕量化套件，整合文字編輯器功能，並針對全站進行 SEO 關鍵字結構優化。
極致 RWD 校正
針對不同裝置尺寸進行像素級的仔細校正，確保在 150 頁的繁雜資訊下，用戶在任何載具皆能獲得最精準、流暢的瀏覽體驗。

學術/技術服務預約平台
國科會自然處尖端核心設施

網址：https://www.acoreplus2024.com.tw/
一個結合學術權威感與自動化技術申請流程的整合型專案。負責品牌視覺的高度還原，專注於複雜的「技術服務預約」與「訂單申請」流程，轉化為流暢且直覺的前端交互體驗。
專業服務流程的 UI 轉譯
將研究所繁複的技術申請表單與規範，實作為簡潔、易操作的功能介面。透過嚴謹的前端邏輯引導，確保使用者在進行技術服務「下單」與「申請」時，能精準完成各項細節，減少溝通成本。
品牌權威與功能的結合
在維持研究單位高質感視覺（如層次感排版、動態進場）的同時，完美整合了會員管理系統與訂單追蹤功能，讓嚴肅的學術服務也能具備現代化的數位操作質感。
高精度的 RWD 實作
由於申請流程包含大量表單欄位，針對各類裝置進行了精細的校正，確保使用者在不同載具都能精準、無誤地完成申請程序。

工業形象官網
LCS Auto & Hardware

網址：https://www.lcsdevco.com.tw/
精準控制工業產品影像的呈現與 RWD 佈局，確保型錄資訊在各種裝置上皆具備極佳的讀取性與載入速度。在這個專案中，我運用了大量的 GSAP (GreenSock Animation Platform) 技術，將剛硬的工業產品轉化為具備高質感與流動感的數位體驗。
GSAP 驅動的動態交互（抓住眼球）
首頁產品區實作垂直式動態圖文切換效果，透過 GSAP 精準控制捲動節奏，創造出層次分明的進場動態，成功提高使用者的停留時間與下滑意願，將傳統的工業產品介紹轉化為震撼的視覺敘事。
導覽體驗優化（增加互動感）
在產品介紹頁面中，規劃了產品項目錨點定位，讓客戶能根據需求快速跳轉至目標產品，並在定位過程中維持優雅的過場視覺，在提升操作便利性的同時，也為嚴謹的工業型錄增添了細膩的互動感。
性能與視覺的平衡
雖然使用了大量的動態效果，但我針對 GSAP 的渲染路徑進行了優化，確保在呈現高解析度工業影像時，依然能維持極佳的流暢度與跨瀏覽器相容性。
